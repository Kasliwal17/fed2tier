*********
Tutorials
*********

**Fed2Tier** allows you to do federated learning in real time on various supported edge devices, including Intel CPUs, Nvidia GPUs, Nvidia Jetson, Raspberry Pi, Intel NUC. **Fed2Tier** provides modular tools and standard algorithms to simplify federated learning implementation in real time using gRPC framework.
